A JSON Web Token (JWT) Library

Please, see our documentation and guides [here](http://hexdocs.pm/joken/)

## Usage

Add Joken to your deps:

``` elixir
def deps do
  # .. other deps
  {:joken, "~> 2.0-rc0"}
end
```

All set!
